Компания АО «Спарго Технологии» провела работы по модернизации программного обеспечения для реализации требований постановления правительства РФ №62 от 27.01.2017 «О проведении эксперимента по маркировке контрольными (идентификационными) знаками и мониторингу за оборотом отдельных видов лекарственных препаратов для медицинского назначения».

В обновленной версии «F3TAIL» (версия 3.3.) доступны новые функциональные возможности:

  • контроль на приходе (регистрация);
  • контроль при подборе товара в чек/расходную накладную (валидация);
  • контроль на расходе (фиксация выбытия номенклатуры);
  • теперь «АРМ Кассира» и «F3TAIL» поддерживают подбор товара по двумерному штрих-коду формата DATAMATRIX. Уточняем, что подбор маркированных позиций возможен только через сканирование двумерного штрих-кода.
пример вызова кода для авторизации в приложении

В переходный период схемы обмена данными могут неоднократно меняться, поэтому особое внимание нужно уделить выбору программного обеспечения, которое легко адаптируется под необходимые нововведения. Решения АО «Спарго Технологии» помогут вовремя и с минимальными затратами начать работу в соответствии с новыми требованиями закона.

Начать тестирование обновленного программного обеспечения клиенты компании АО «Спарго Технологии» смогут начать уже сегодня на реальных маркированных упаковках. Для этого необходимо:

  • обновить программное обеспечение;
  • зарегистрироваться в рабочем личном кабинете;
  • модернизировать оборудование (ЭЦП для работы с ИС «Маркировка», сканер с возможностью считывания штрих-кода формата DATAMATRIX).
